# MS-Digital-Wallet
Projeto de aprendizado focado na construção de uma Carteira Digital com arquitetura de microserviços em Go. O objetivo é praticar conceitos de sistemas distribuídos, banco por serviço, comunicação síncrona (HTTP/JSON) e assíncrona (event-driven), isolamento de serviços e orquestração de pagamentos em um cenário inspirado em fintech.
